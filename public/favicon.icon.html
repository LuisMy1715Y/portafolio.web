Cómo usarlo en Next.js

Guarda este archivo como favicon.svg en la carpeta public/ de tu proyecto y agrégalo en tu next.config.js o en _app.tsx usando:

tsx
import Head from 'next/head';

export default function MyApp({ Component, pageProps }) {
  return (
    <>
      <Head>
        <link rel="icon" href="/favicon.svg" type="image/svg+xml" />
      </Head>
      <Component {...pageProps} />
    </>
  );
}
